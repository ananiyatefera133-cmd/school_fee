# School Fee Management Website

A basic school fee management system built with:
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP (with SQLite)

## Features
- Navigation menu
- Registration page
- Login page
- Home/index page
- Dashboard with fee summary and records
- Image and video section on the home page

## Run locally (required for PHP pages)
```bash
php -S 0.0.0.0:8000
```
Then open:
- `http://localhost:8000/index.php` (full PHP app)

## Preview troubleshooting (Not Found)
Many IDE/repo preview panes are **static** and do not execute PHP.
- Open `index.html` for static preview.
- Do **not** open `index.php` in static preview mode.
- For full functionality, run PHP server and open `http://localhost:8000/index.php`.
